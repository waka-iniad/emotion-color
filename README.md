# emotion-color


# 感情から色を生成するアートプロジェクト

## 概要
このプロジェクトは、入力した感情キーワードから対応する色を表示するシンプルなPythonアプリです。  
感情に合った色をビジュアル化し、「感情の色」を楽しめるインタラクティブなアート作品を目指しています。

## 技術スタック
- Python
- Streamlit または Jupyter Notebook
- 感情分析ライブラリ（例: transformers）
